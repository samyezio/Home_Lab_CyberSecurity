# CyberLab (Home Lab Cybersécurité)

## Objectif

Home lab isolé et reproductible sous VMware (Kali + machine cible) pour pratiquer Nmap/Wireshark et documenter les résultats.

## Structure

- docs/ : topologie, notes

- reports/ : rapports (Nmap, Wireshark)

- pcap/ : captures Wireshark

- screenshots/ : preuves (captures d’écran)

- scripts/ : scripts d’automatisation

## Progression

- Day 1: setup lab + réseau + snapshots + topologie
- Day 2: Nmap Scanning
